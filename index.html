Me apasiona la idea de poder crear soluciones tecnológicas que mejoren la vida de las personas y que contribuyan al desarrollo de la sociedad. Estoy siempre en busca de nuevos retos y oportunidades para seguir aprendiendo y creciendo como profesional.

Metacalor

Aplicación web para el control y registro de caloríco del usuario. Permite seleccionar alimentos desde una base de datos con información nutrimental detallada y también añadir alimentos personalizados. La app facilita el monitoreo diario de ingesta calórica y fue desarrollada para brindar un enfoque flexible y preciso en el seguimiento nutricional dando advertencias y notificaciones cuando se estaba a punto de llegar al límite de calorías por día o si se excedía dicho límite.

PHP React Bootstrap MySQL

FitnessTracking

Sistema de ejercicio con rutinas personalizadas para trackeo del cuerpo integrada con una API REST a una app móvil. Utiliza un algoritmo KNN para crear la rutina del usuario en base a sus características y objetivos. Además, utiliza la tecnología de Google MediaPIPE para crear modelos de maching learning que identificaran los ejercicios a realizar para validar las repeticiones del usuario.

Ganó mención honorífica en el concurso de innovación tecnológica del Centro Universitario de Ciencias Exactas e Ingenierías.

Plataforma de Blogs de Lucha Libre

Sitio web dinámico y responsivo desarrollado con Laravel y Tailwind CSS, enfocado en la publicación de contenido sobre lucha libre. Permite a los usuarios autenticarse, crear, editar y eliminar publicaciones según su rol.

Escritores: pueden crear y editar sus propias publicaciones.

Administradores: poseen control completo sobre todas las entradas y gestión de usuarios (crear, editar y eliminar).
