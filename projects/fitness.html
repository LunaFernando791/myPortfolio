FitnessTracking

Description

FitnessTracking is an innovative exercise tracking system designed to generate personalized workout routines and monitor user performance through advanced machine learning and computer vision. The system integrates a RESTful API built with Laravel to communicate seamlessly with a mobile application developed in Kotlin, using Jetpack Compose for a modern and responsive UI experience.

At its core, the app uses a K-Nearest Neighbors (KNN) algorithm to tailor workout plans based on each user's physical characteristics and fitness goals. To validate exercise execution and count repetitions, the system leverages Google MediaPipe, enabling real-time pose estimation and activity recognition through custom machine learning models.

To enhance user engagement and retention, FitnessTracking incorporates gamification mechanics such as achievement badges, level progression, and global rankings. These features encourage consistent participation and foster a sense of community among users.

The backend infrastructure is powered by Laravel and MySQL , ensuring robust data management and scalable API performance. The project also utilizes Bootstrap for web-based components and AWS for cloud deployment and media handling.
