BrainMed

Description

This project is a lightweight and intuitive web application designed to encrypt and decrypt messages directly in the browser using JavaScript. Built with a focus on simplicity and client-side security, the app allows users to input any text and transform it into an encoded format that can be safely shared or stored.

The interface is developed using HTML5 and CSS3, ensuring a responsive and clean user experience across devices. The core encryption logic is implemented in vanilla JavaScript, leveraging string manipulation and basic cryptographic techniques to obfuscate the message content. While not intended for high-security applications, it serves as an educational tool and a practical demonstration of how encryption can be handled on the frontend.
